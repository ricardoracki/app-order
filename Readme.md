# Order

App de lista de compras

### Iniciando a aplicação

Execute o comando para iniciar o aplicativo

```bash
npm start
```

### Stack

- React Native
- Expo
- Typescript
- zustand
- AsyncStorage
- Offline

### Features

- Criar uma lista de compras organizada
- Fácil interação
- Envio da lista para outros usuários
- E mais...
